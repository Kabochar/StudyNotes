## 总结

代码规范性对于 Web 安全来说非常重要，尽量不要采用直接拼接的方式进行查询。

在 Web 上线之后，还需要将生产环境中的错误提示信息关闭，以减少被 SQL 注入的风险。

也可以采用第三方的工具，比如 SQLmap 来对 Web 应用进行检测，以增强 Web 安全性。
